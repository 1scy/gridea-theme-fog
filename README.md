模型下载地址：[戳此](https://gitee.com/ericam/live2d-widget-models )

模型样式预览：[戳此](https://blog.csdn.net/wang_123_zy/article/details/87181892 )

下载完后挑选一个自己喜欢的模型，复制assets文件夹替换到fog\assets\media\live2d路径下的assets文件夹即可。
